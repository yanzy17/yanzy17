🔄 Processing Flowchart

🧭
GTD Processing Steps:
For each item above, ask: "Is this actionable?"
✅ Actionable → Project Tracker

• Follow up with Acme (→ add to Projects)

• Team morale activity (→ add to Projects)

• Book dentist (→ add to Today's Tasks)

• Birthday gift for mom (→ add to Calendar)

📚 Reference → Resources

• Async communication article (→ save to Resources)

• Research Rust (→ save to Learning list)

💭 Someday/Maybe

• Habit tracker app idea (→ Someday list)

🗑️ Trash

Nothing to delete this time!

🔄 GTD Decision Tree

1. What is it? → Define the item clearly

2. Is it actionable? → If NO: Trash / Reference / Someday

3. If YES: What's the next action?

• Takes <2 min? → Do it NOW

• Am I the best person? → If NO: Delegate

• Specific time needed? → Calendar

• Otherwise → Add to Next Actions list

4. Part of a bigger outcome? → Create/update Project

⚡ Formulas & Automations

Copy-paste these Notion formulas into your databases

⚠️
How to use: In Notion, create a Formula property in your database, then paste the formula code below. Make sure your property names match exactly.

1. 🏆 Productivity Score

Used in: Daily Planner database

if(prop("Deep Work Hours") >= 3 and prop("Mood") == "😊Great", "🏆 Great Day", if(prop("Deep Work Hours") >= 2, "👍 Good Day", "🔄 Needs Improvement"))

2. 📅 Days Until Due

Used in: Project Tracker database

dateBetween(prop("Due Date"), now(), "days")
💡
Returns negative number if overdue. Combine with conditional formatting to highlight urgent items.

3. 🔥 Streak Counter Logic

Used in: Habit Tracker (explanation — requires Notion relations)

How streak counting works in Notion:

1. Create a "Habit Log" database with: Date, Habit Name, Completed (checkbox)

2. Create a relation from Habit Tracker → Habit Log

3. Use a Rollup to count consecutive "Completed = true" entries

4. For accurate streak counting, use Notion automations or a formula:

if(prop("Today Completed") and prop("Yesterday Completed"), prop("Previous Streak") + 1, if(prop("Today Completed"), 1, 0))

4. 📊 Progress Percentage

Used in: Project Tracker (when using sub-tasks)

round(prop("Tasks Done") / prop("Total Tasks") * 100)

5. 📆 Week Number

Used in: Any database with dates

formatDate(prop("Date"), "W")

6. 🚦 Auto-Status Based on Progress

Used in: Project Tracker (alternative to manual status)

if(prop("Progress") == 100, "✅ Done", if(prop("Progress") > 0, "🔄 In Progress", if(prop("Due Date") < now(), "🚨 Overdue", "📥 Not Started")))

7. ⏰ Meeting Load Warning

Used in: Daily Planner

if(prop("Meetings Count") >= 5, "🚨 Too Many!", if(prop("Meetings Count") >= 3, "⚠️ Heavy", "✅ Manageable"))

8. 📈 Weekly Deep Work Total

Used in: Weekly Planner (rollup from Daily Planner)

// Use a Rollup property: // Relation → Daily Planner // Property → Deep Work Hours // Calculate → Sum
🔮
Pro Tip: Notion's new formula 2.0 supports if/else, variables, and more complex logic. Check notion.so/help/formulas for the latest syntax reference.
🚀 Hybrid Worker Pro / 🚀 Quick Start Guide

🚀 Quick Start Guide

Get up and running with your Hybrid Worker Pro workspace in 7 days

⏱️
Total setup time: ~2 hours spread over 7 days. After that, just 5 minutes each morning and 2 minutes each evening to maintain.

📋 How to Duplicate This Template

Option A — Direct Duplicate:

  1. Open the shared Notion page link
  2. Click "Duplicate" in the top-right corner
  3. The entire workspace copies to your Notion account
  4. Customize properties, remove sample data, add your own

Option B — Build from Scratch (using this guide):

  1. Create a new page in Notion called "Hybrid Worker Pro"
  2. Add sub-pages for each section (Dashboard, Daily Planner, etc.)
  3. Create databases with the properties listed in each section
  4. Add the formulas from the ⚡ Formulas page
  5. Set up your views (Table, Board, Calendar)

📅 First 7 Days Setup Plan

▶ Day 1: Foundation (30 min)
Duplicate template or create main page structure
Set up Daily Planner database with all properties
Fill in today's entry as practice
▶ Day 2: Planning (20 min)
Set up Weekly Planner with your actual schedule
Define your WFH vs Office days
Create time block templates for each day type
▶ Day 3: Projects (20 min)
Set up Project Tracker database
Add all current projects (work + personal)
Assign priorities and due dates
▶ Day 4: Habits (15 min)
Choose your 7 daily habits (start with 3-5 if new)
Set up Habit Tracker grid
Start tracking today
▶ Day 5: Meetings (15 min)
Set up Meeting Notes database
Create meeting template
Log your next meeting using the template
▶ Day 6: Resources (15 min)
Set up Resources database
Add 5-10 tools/articles you already use
Set up Brain Dump page for quick capture
▶ Day 7: Review & Customize (20 min)
Do your first Weekly Review
Set up Dashboard with linked databases
Customize colors, icons, and layout to your preference
Set reminder: Weekly Review every Friday 4pm

🎨 Common Customizations

🎯 Adjust to your style:

  • Change habit list to match your goals (start with 3-5 habits, not 7)
  • Modify time blocks to match your work schedule
  • Add/remove database properties as needed
  • Create different views: Calendar for deadlines, Board for projects
  • Add relations between databases (Daily Planner ↔ Projects)
  • Set up Notion automations for recurring tasks
  • Create a "Favorites" section in sidebar for most-used pages
