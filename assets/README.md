## 3️⃣ NOTION TEMPLATE — `assets/notion/ai-solopreneur-command-center.md`

**Apa ini:** Struktur lengkap "AI Solopreneur Command Center" dalam format Notion-compatible Markdown. Berisi 8 page utama:

- 🏠 Mission Control (Dashboard)
- 🧠 The Brain (Knowledge Base)
- 💰 Revenue OS
- ⚙️ Automation Lab
- 📣 Content Engine
- 👥 Client CRM
- 📊 Metrics & Money
- 📚 Resource Vault
- 🎁 Prompt Library (50+ prompts)

**Cara setup di Notion:**

1. Buka [notion.so](https://notion.so) → buat Workspace baru (atau gunakan yang ada).
2. Buat halaman utama → klik `/` → pilih `Import` → `Markdown`.
3. Upload file `.md` → Notion akan auto-format jadi nested pages.
4. Sesuaikan visual: tambah cover image, icon, ubah view database (Kanban, Calendar, Gallery).
5. Tambahkan **3-5 baris contoh** di setiap database (sample data sudah saya tulis di file).

**Cara distribusi ke pembeli:**

1. Klik tombol `Share` di kanan atas halaman utama.
2. Toggle `Allow duplicate as template`.
3. Toggle `Share to web`.
4. Copy URL yang berakhiran `?duplicate=true`.
5. URL ini yang Anda kirim ke pembeli (auto-deliver via Gumroad).

**Tip:** Buat akun Notion terpisah khusus master template, agar Anda bisa update setiap quarter tanpa mengganggu data pribadi.

## 4️⃣ EMAIL SEQUENCE — `assets/marketing/email-launch-sequence.md`

**Apa ini:** 7 email pre-launch sequence (T-7 sampai launch+1). Setiap email punya 3 variasi subject line untuk split-test.

**Email 1** — T-7: "Something is broken in your business"
**Email 2** — T-5: "How I escaped"
**Email 3** — T-3: "Behind the scenes (the only 4 AI tools you need)"
**Email 4** — T-2: "The 7 automations"
**Email 5** — T-1: "Doors open in 24 hours"
**Email 6** — Day 0: "It's live"
**Email 7** — Day +1: "Last call at $47"

**Cara setup di Beehiiv:**

1. Daftar [beehiiv.com](https://beehiiv.com) (gratis sampai 2,500 subscriber).
2. Buat publication → setup branding (logo, warna, font).
3. Buat **Automation** → "Welcome Sequence."
4. Copy-paste isi email 1-7 ke automation builder.
5. Set delay: Email 1 (Day 0), Email 2 (Day +2), Email 3 (Day +4), dst.
6. Aktifkan automation → arahkan semua subscriber baru ke sini.

**Cara setup di ConvertKit / Mailchimp:**
Sama persis — semua platform email punya fitur "automation" / "drip campaign."

**Variabel yang harus diganti:**
- `[FIRST NAME]` → tag personalisasi platform (`{{first_name}}` di Beehiiv)
- `[LANDING PAGE LINK]` → URL landing page Anda
- `[BUY IT NOW]` → URL Gumroad
