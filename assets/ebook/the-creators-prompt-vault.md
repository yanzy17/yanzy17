# HOW TO USE THIS VAULT — READ FIRST

These 100 prompts are tested across ChatGPT-4o, Claude 3.5+, Gemini, and Perplexity. Each one is built around the same **5-part architecture**:

```
1. ROLE   — who the AI is
2. TASK   — what to do
3. INPUT  — what you're giving it
4. RULES  — constraints to prevent generic output
5. FORMAT — exactly how to return the result
```

**The 3-step workflow:**

1. **Pick** — find the right prompt for the task at hand.
2. **Fill** — replace every `[BRACKET]` with your real input. Don't skip any.
3. **Refine** — if output isn't perfect, follow up with: *"Now make it sharper / shorter / more like [example]."*

> **Pro move:** Save your top 10 most-used prompts as **ChatGPT custom instructions** or **Claude projects**. You'll save another 4 hours/week.

# SECTION 1 — HOOK GENERATORS (10)

### Prompt #1 — Multi-Hook Generator
**Use when:** You need 10 hook options for a post.
```
Generate 10 hooks for a [PLATFORM] post about [TOPIC].

Mix these formulas:
- Contrarian ("Everyone says X. They're wrong.")
- Number-led ("3 mistakes [audience] makes")
- Curiosity gap ("The reason most X fail isn't what you think")
- Specific story ("I lost $X doing Y")
- Prediction ("In 12 months, X will be obsolete")

Constraints:
- Each hook ≤ 12 words
- No emoji on first line
- Plain English (no "game-changer", "unlock", "leverage")

Output as numbered list.
```

# BONUS — THE 5 PROMPT ENGINEERING RULES

### Rule 1 — Specificity beats cleverness
Bad: "Write me a tweet about marketing."
Good: "Write a 240-character tweet for SaaS founders about why CAC payback matters more than LTV. Voice: sharp, contrarian. No emojis."

### Rule 2 — Constraints make better output
The more rules you add, the less generic the answer. AI loves to be average. Constraints force it to commit.

### Rule 3 — Show, don't tell
"Write in my voice" → mediocre.
"Write in my voice. Here are 5 examples of my writing: [paste]" → 10x better.

### Rule 4 — Format your output
Always end with: "Output as: [bulleted list / table / code block / numbered with labels]". Otherwise AI defaults to wall-of-text.

### Rule 5 — Iterate, don't restart
If output is 70% there, follow up: *"Make this sharper"* or *"More like example 2"*. Don't start a new prompt — refine.
